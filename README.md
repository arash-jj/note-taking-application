# Note Taking Application

A full-stack note taking web application with a React/TypeScript/Vite frontend and a Node.js/Express/TypeScript backend using MongoDB. Users can create, edit, archive, tag, search, and manage rich-text notes with authentication.

---

## 🚀 Getting Started

This repository contains two main projects:

| Folder      | Description                                    |
| ----------- | ---------------------------------------------- |
| `backend/`  | RESTful API server (Express + TypeScript)      |
| `frontend/` | Client application (React + Vite + TypeScript) |

You can develop them separately or run both side-by-side for the full experience.

### Prerequisites

- Node.js v16+ / npm
- MongoDB (local or cloud URI)

### Workspace Setup

Open the root of the repository in VS Code, then:

```bash
# install dependencies for both projects
cd backend && npm install
cd ../frontend && npm install
```

Each project has its own README with detailed setup instructions (see below).
